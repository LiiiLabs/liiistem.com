Liii STEM v2025.1 Version Change Log ​

⭐️ New Features and Highlights ​

  1. AI streaming output + toolbar entry: AI features now stream responses and are accessible from a dedicated toolbar entry for smoother interaction.
  2. LLM upgrade + new model support: Upgraded to v20251024, with new support for DeepSeek 3.2 and Qwen 3.
  3. Handwriting / image OCR: Added an OCR model for handwritten formulas and image recognition.
  4. Comments: Insert comments and set comment authors (Packages → Utilities → Comment, Ctrl+/ to insert), plus fold/unfold with Alt+Shift+↑/↓.
  5. Emoji quick input: e.g. type :anger: to insert 💢.
  6. Custom start number for ordered lists: set the starting index via item-nr, and keep numbering independent across lists.
  7. Statistics window improvements: more complete info, with better layout and sizing.
  8. Themes & UI: added Liii Dark theme, icon polish, and fixes for dark theme and course style buttons.
  9. Editing & typesetting: smarter Tab completion, more shortcuts, richer alignment/list styles, and better math font/symbol support (incl. \mathscr).
  10. Export & stability: PDF export supports Emoji, better image embedding, and multiple key stability fixes.

🌟 Summary ​

Liii STEM v2025.1 focuses on productivity and polish: faster editing workflows, stronger math typesetting, a more complete comment system, expanded AI/OCR capabilities, improved PDF export, and many stability and cross-platform fixes.

🔧 Changelog ​

Liii STEM v2025.1.6 (2025.11.07) ​

  1. Ordered lists now support custom starting numbers (item-nr), and numbering stays independent across list environments.
  2. Added comment insertion and author settings (Packages → Utilities → Comment), shortcut Ctrl+/ to insert; comments can be folded/unfolded with Alt+Shift+↑/↓ (added in v2025.1.5 and highlighted again here).
  3. Added the “does not divide” symbol ∤ to the Math Mode toolbar, with shortcut | then /.
  4. In the environment toolbar, “Preferences for tag” now supports switching chapter/section numbering formats.
  5. Character counting now ignores line breaks.
  6. In the liii-night theme, removed redundant indent/alignment buttons from the source-style toolbar.
  7. Fixed button colors for course style in dark mode.
  8. Fixed tooltip localization for “Insert ellipse”.
  9. Improved localization coverage for math superscript/subscript menu items and related task-document guidance.
  10. Localized the dialog for Edit → Search recent documents.
  11. Translation tweaks: “alignment equation” → “multi-line equation (align)”, “several equation” → “multi-line equation (eqnarray)”.
  12. Prevented session menu items from being translated, keeping technical terms consistent.
  13. Adjusted circled-number alignment: replaced aligned-space-item with aligned-item to avoid centering issues.
  14. Fixed truncated text in the input box of “Search recent documents”.
  15. Fixed navigation commands (next/previous) unexpectedly changing the converter structure; now preserves the user-selected converter state.

Liii STEM v2025.1.5 (2025.10.23) ​

  1. Improved the Statistics window (open via Tools → Statistics) with more complete information.
  2. Optimized Statistics window styling: adjusted window width and data spacing.
  3. Fixed a critical issue where Beamer mode couldn’t switch pages (a regression from large-document optimizations).
  4. Added macOS trackpad zoom support.
  5. Fixed \mathcatalan incorrectly rendering in italic; restored upright “C”.
  6. Added shortcut inputs for some Emoji.
  7. Upgraded LLM to v20251024, removing some legacy models.
  8. Added support for DeepSeek 3.2 and Qwen 3.
  9. Added an OCR model with built-in system prompts for handwritten formulas / image recognition.
  10. Added comment insertion and author settings (Packages → Utilities → Comment), shortcut Ctrl+/ to insert.
  11. Comments can be folded/unfolded with Alt+Shift+↑/↓.
  12. Added a shortcut for the align environment: Ctrl+$ (also added an option in the menu bar).
  13. Emoji shortcut example: in Math Mode, type :anger: to insert 💢.
  14. Fixed a plugin issue where continuous input kept stealing focus and the table of contents couldn’t jump correctly.

Liii STEM v2025.1.4 (2025.10.11) ​

  1. Fixed PDF export failing with Emoji; PDF export now supports Emoji.
  2. Supports dragging images from the browser to embed (WebP not supported yet).
  3. Added a “123” numbering button to the align toolbar, with shortcut Ctrl+#.
  4. Presentation mode (View) now supports Esc to exit fullscreen.
  5. Math Mode toolbar accent icons switched from PNG to SVG.
  6. Read-only files now show a save prompt and a [Read-only] label; saving prompts “Save as”.
  7. Fixed comment feature enablement: the comment menu can be enabled via Document → Style → Packages → Utilities → comment.
  8. Improved the dialog when closing buffered files to a “Save / Don’t save” flow.
  9. Known issue: when multiple temporary files exist, choosing “Save” only saves the current file; closing an existing file can still save modifications.
  10. Fixed a ghosting issue where </> + Tab showed <> in Math Mode.
  11. Added MATLAB syntax highlighting: Document → Style → Packages → Code → matlab; create via Insert → Macro → matlab. Use matlab inline and matlab-code for code blocks.
  12. Reduced view-zoom lag for large documents.
  13. Preserves cursor position when switching between body and preamble.
  14. Adjusted circled-number shortcuts to @ + number + Tab (0-20) to avoid accidental triggers in email addresses.
  15. Added parenthesized-number shortcut: ( + number + Tab.
  16. Added Roman numeral shortcuts: I 5 Tab → Ⅴ, i 5 Tab → ⅴ.
  17. Added ordered-list styles: Chinese numerals, circled numbers, and parenthesized numbers.
  18. Adjusted ordered-list letter styles to a. b. c. and A. B. C..
  19. Added menu entries for ordered lists/arrangements: Insert → Ordered list … / Insert → Ordered arrangement ….
  20. Optimized arrangement line spacing (smaller default).
  21. Fixed gaps between table rows when tables break across pages.

Liii STEM v2025.1.3 (2025.09.15) ​

  1. Fixed a freeze when inputting an ellipsis via Shift+6.
  2. Fixed unresponsiveness caused by multiple copy operations after selecting all.
  3. Fixed PDF export failure when image content and format didn’t match.
  4. Fixed the Gnuplot plugin not rendering correctly inside folded structures.
  5. Windows installer now uses a code-signing certificate.
  6. Added a dedicated installer for macOS 12.
  7. Moved header/footer features into an auxiliary window.
  8. Fixed duplicate submenu items for numbered single-line formulas.
  9. Restored slide order adjustment buttons.
  10. Supports embedding linked images, or dragging images directly into documents.
  11. When selecting the dark theme, the app now prompts for restart and offers one-click restart.
  12. Added friendlier dialogs when opening unreadable files and read-only files.
  13. Added a success notification after PDF export completes.
  14. Supports English hyphens in Chinese documents.
  15. Updated table and matrix insertion shortcuts.
  16. Added shortcuts for inserting additional math fonts.
  17. Added quick input support for multiple integral symbol variants.
  18. Added \mathscr math font support.
  19. Added an “Appendix” insertion option in the mode toolbar.
  20. Improved text highlight rendering.

Liii STEM v2025.1.2 (2025.08.28) ​

  1. AI now supports streaming output, showing generation progress in real time.
  2. AI features are integrated into a dedicated toolbar icon entry.
  3. Theme updates: icon polish for the classic Liii theme, plus a new Liii Dark theme.
  4. Search & replace improvements: structured search/replace window embedded as an auxiliary window, supporting complex structures (e.g. formulas) via the Mode Toolbar.
  5. Added Tab completion in the editing window, covering text, formulas, macro commands, and more.
  6. macOS shortcuts: Cmd+T for new tabs, Cmd+1-6 to switch tabs.
  7. Multi-tab management improvements: added window attach/detach management and fixed many compatibility issues.
  8. Improved file format option display in Save / Save As / Open dialogs.
  9. Comprehensive performance optimizations: faster startup, more responsive editing, smoother rendering.
  10. Math Mode multiplication symbol defaults to ×, and Tab can cycle through other multiplication symbols.
  11. Added multi-integral macro support: \iint, \oiint, \iiint, \oint, \idotsint, etc.
  12. Magic Paste supports Emoji (beta) and smarter detection of more content types.
  13. Enunciation environments now use European-style numbering (independent numbering per environment type).
  14. Added the Assumption enunciation environment.
  15. Fixed an issue on Windows where images on the D: drive couldn’t be inserted.
