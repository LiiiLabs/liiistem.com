Inserting Ordered/Unordered Lists ​

Liii STEM provides the function of inserting numbered ordered lists and unnumbered unordered lists. This guide will help you use lists to achieve the standard typesetting shown in the figure below.

1 Basic Methods for Inserting Ordered Lists ​

1.1 Inserting Ordered List Environment ​

You can insert ordered list environment through the following three methods:

  • In the menu bar, click Insert → Ordered List. You can choose different number or letter numbering.

  • In Text Mode, click the button in the Mode Toolbar to insert the required type of list.

  • Use shortcut 1 → . → Tab. This will directly enter the default ordered list environment with Arabic numeral numbering.

1.2 Editing Text in Ordered List Environment ​

After entering the ordered list environment, Liii STEM will display the following interface:

1.2.1 List Environment and Item Sub-environment ​

The environment toolbar shows that the current cursor is in a sub-environment named item, and an entry numbered with Arabic numeral 1 has already started.

  • You can edit the content you need in this sub-environment (i.e., entry).
  • Use the Enter key on Windows or return key on macOS to jump to the next entry for editing.
  • Liii STEM will automatically number using corresponding numbers or letters.

Usage effect as shown below:

After returning from the item sub-environment to the ordered list environment, the environment toolbar will change.

1.2.2 Changing Numbering Type ​

In the list environment, you can click the "Structure Variant" option in the environment toolbar to change the numbers or letters used for numbering. For example, changing Arabic numeral numbering to lowercase Roman numeral numbering.

1.2.3 Converting to Unordered List ​

In the list environment, you can also click the "Toggle numbering" option in the environment toolbar to convert it to an unordered list.

2 Basic Methods for Inserting Unordered Lists ​

Unordered list usage methods are basically the same as ordered lists.

2.1 Entering Unordered List Environment ​

Similarly, you can enter the unordered list environment through the following three ways:

  • In the menu bar, click Insert → Unordered List. You can choose different symbol markers.
  • In Text Mode, click "Insert List" in the Mode Toolbar, similar to the way of inserting ordered lists.
  • Use shortcut + → Tab. This will directly enter the default unordered list environment with dot markers.

2.2 Editing Text in Unordered List Environment ​

After entering the unordered list environment, the cursor is now in a sub-environment named item, and an entry marked with a dot has already started. You can edit the content of this entry. After completing this entry, you can use the Enter key on Windows or return key on macOS to jump to the next entry.

Usage effect as shown below:

After returning from the item sub-environment to the unordered list environment, you can click the "Structure Variant" option in the environment toolbar to change the symbols used for marking, or click the "Toggle numbering" option in the environment toolbar to convert it to an ordered list.

3 Nesting of Ordered and Unordered Lists ​

Ordered list environments and unordered list environments support nested usage. Liii STEM will automatically handle different levels of indentation, numbering, and marking. Usage effect as shown in the figure.
