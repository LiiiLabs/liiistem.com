Table Numbering and Referencing in Liii STEM ​

I. Feature Introduction ​

This feature provides automatic numbering and cross-referencing for tables with bidirectional linking.

II. Instructions ​

STEP 1. Insert Table ​

  • Click sequentially in the toolbar: Insert -> Table -> Small Table (as shown in the image below)
  • Click the up, down, left, and right arrows with "+" or "×" on the toolbar to add or delete rows and columns (as shown in the image below)

Note: Generally, the system will automatically number your table after insertion. If you find it's not numbered, you can place the cursor on the table and click the Ⅳ style button on the toolbar (as shown in the image below) to implement it manually.

STEP 2. Table Naming ​

  • Place the cursor at the end of the table caption and ensure the input method is in English mode
  • Click \, type label, (or use the shortcut: ctrl+!), press Enter
  • Enter your table name, such as "Table1", press Enter (as shown in the image below)

STEP 3. Establish Bidirectional Link ​

  • At the place where you need to reference this table
  • Click \, type ref, press Enter
  • Enter the table name you just named again, press Enter (as shown in the image below)

STEP 4. Insert Non-breaking Space ​

  • Place the cursor at the position where you want to insert
  • Hold Space, press Tab (as shown in the image below)

Note: Non-breaking Space ensures that the front and back parts are on the same line and will not be separated due to line breaks.
