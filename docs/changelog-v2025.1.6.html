Liii STEM v2025.1.5 & 1.6 Released! ​

Liii STEM

It is an editor designed for STEM fields, integrating:

  • Equation editing: Type any math symbol in seconds with our unique system.

  • Auto-formatting: Just input your content -- Liii STEM auto-generates structured handouts and notes that meet academic standards. No more layout headaches.

  • AI features: Liii STEM's built-in LLMs supports not only plain text, but also math formulas, tables, and even photos of handwritten notes. All AI-generated content is directly formatted inside Liii STEM.

  • Code execution: Liii STEM formats code automatically -- handling fonts, colors, and indentation for clean source code presentation.

  • Magic Paste: Paste content from ChatGPT, DeepSeek, etc. -- it auto-formats to save you layout work.

  • Premium Service: As the commercial edition of Mogan STEM, Liii STEM offers not only AI features but also up to two hours of weekly one-on-one support. We also provide custom templates and article revision services. Details: https://liiistem.com/pricing-fruit.html

Faster, neater, and more efficient than handwriting! Liii STEM is very easy to use. Here's an example of notes written using Liii STEM by a statistics student at Southern University of Science and Technology:https://liiistem.com/library/notes/STAT850.html.

New Trackpad Support, Comments feature & Upgraded LLM! ​

  • LLM: Upgraded to v20251024. Removed some legacy models and added support for DeepSeek 3.2 and Qwen 3.

Added OCR models for handwritten formula recognition and image-based tasks --- no prompts needed.

Added in v2025.1.5.

  • New Comment Feature. Add comment from the focus toolbar: Packages → Utilities → Comment. Then you can insert a comment via Command+/(macOS), Ctrl+: (Windows)

You can customize the comment author in the focus toolbar.

Fold/unfold comments with Alt+Shift+↑/↓.

Added in v2025.1.5.

  • macOS Trackpad Gesture Support Zoom is now supported using macOS trackpad gestures.

Added in v2025.1.5.

New Keyboard Shortcuts & Align Environment! ​

  • Shortcut for Align Environment

Previously, the align environment could only be inserted by the align command.

Now, use the shortcut Ctrl+$ or select it from the menu bar.

Added in v2025.1.5.

  • New Emoji Shortcuts

Added shortcuts for commonly used emojis that previously lacked them.

Example: In math mode, type :anger: to insert 💢.

Added in v2025.1.5.

  • Custom Start Number for Enumerate Lists

Select any enumerate list and set its starting number in the Focus toolbar: Rendering options for tag.

Added in v2025.1.6.

  • "Does Not Divide" Symbol Added & Shortcuts Optimized

The "does not divide" symbol ∤ has been added to the math mode toolbar.

Use the shortcut |+/ to insert it quickly.

Added in v2025.1.6.

  • Section Numbering Format Control

Customize section numbering formats via the focus toolbar: Preferences for tag.

Added in v2025.1.6.

  • Line Breaks Excluded from Character Count

Character count now excludes line break characters.

Added in v2025.1.6.
