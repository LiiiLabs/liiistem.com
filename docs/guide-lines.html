Labeling and Referencing Numbered Content in Liii STEM ​

In Liii STEM, you can insert independent formulas, images, tables, and different levels of sections and other environments, and easily achieve automatic numbering of the above content, labeling to ensure automatic updating of serial numbers, and convenient referencing in the following text.

We have also added the smart reference (smart-ref) function to help you automatically identify the type of content you want to reference, achieving referencing of all types of content with one command; the reference preview (preview-ref) function allows you to preview the content you referenced in the preceding text at any time.

The numbered environments supported by Liii STEM include: images, tables, ordered lists, sections, subsections, all enunciations, displayed formulas, multi-line formulas.

1 Basic Methods of Numbering and Labeling ​

1.1 Content Insertion ​

You can insert the content you need in the box by clicking buttons on the Menu Bar or Mode Toolbar. The following will introduce the above three methods in order.

Method 1: Menu Insertion ​

Click Insert in the Menu Bar, select Section / Table / Image / Formula and click, then input or paste your content in the blue box.

Method 2: Mode Toolbar Insertion ​

You can directly select the environment you want to insert in the Mode Toolbar under Text Mode, see the figure below for details.

Note! When inserting tables and images, please make your selection within the red box in the figure below, do not click other buttons.

Environment Insertion Effect Display ​

1.2 Content Numbering ​

Switch to Math Mode, click the 123 style button on the Environment Toolbar to number. Generally, figures, tables, and sections are automatically numbered when inserted; if not automatically numbered, you can click the numbering button to number them yourself.

1.3 Using label Command to Label Content ​

  • At the content you inserted and numbered, sequentially input \ -> label, then press Enter.
  • Input a custom name. It is recommended to use the naming format of "category: name", such as "Fig:model-arch".
  • A blue identifier composed of diamonds and vertical lines will appear to the right of the labeled numbered content.

1.4 Viewing Labels Inserted in Formulas ​

In Liii STEM, when you insert labels in formulas, there are two ways to view the inserted label names:

Through Showing Hidden Content ​

  • Move the cursor to the label position at the end of the formula;

  • Click Focus → Show Hidden Content;

  • The label will be displayed in the editing area.

Through Label Indication Function ​

  • Move the cursor to the label position at the end of the formula;

  • Click Document → Indicate Labels → Detailed;

  • The label (such as eq-square) will be displayed in the upper right corner of the formula.

2 Basic Methods of Referencing ​

Liii STEM provides two referencing methods. You can use the traditional method, which uses the "reference command" to achieve content referencing; or you can use our new feature, "smart reference (smart-ref)", to complete it more efficiently. For details, see section 2 of this guide.

2.1 Using reference Command ​

  • At the place where you need to reference, first input the type name of the content you are referencing, such as "table".
  • Continue to sequentially input \ -> ref, then press Enter.
  • Input the label name of the content to be referenced.

Note: Please ensure your input exactly matches the name you previously customized.

2.2 Inserting Non-breaking Space ​

  • Place the cursor at the position where you want to insert.
  • Hold down the keyboard Space, while tapping Tab.

Note: Non-breaking Space, i.e., "non-breaking space", ensures that the two parts before and after are on the same line and will not be broken by line breaks, improving the standardization of article writing.

3 Using Smart Reference (smart-ref) to Achieve Convenient Referencing ​

Smart reference (smart-ref) can identify the type of content you want to reference and automatically generate it, without you having to manually type the type name and non-breaking space.

3.1 Enabling smart-ref ​

  • Under Text Mode, click the ⊕ button in the Focus Toolbar.
  • In the Utility category, check smart-ref.

3.2 Achieving Quick Referencing ​

  • Sequentially input \ -> smart-ref, click Enter.
  • Input the label name of the content to be referenced, click Enter.

Note: If you need to disable this plugin, you can click the plugin name in the Focus Toolbar under this text mode and select Cancel Package. The preview-ref plugin below can also be canceled in the same way.

4 Using Preview Plugin (preview-ref) to Achieve Reference Preview ​

4.1 Enabling preview-ref ​

  • Under Text Mode, click the ⊕ button in the Focus Toolbar.
  • In the Utility category, check preview-ref.

4.2 Real-time Preview ​

Hover the mouse over the reference number, and you can preview the labeled or referenced content.
