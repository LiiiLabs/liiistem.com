Liii STEM v2025.0 Version Change Log ​

Liii STEM v2025.0.12 (20250716) ​

Update Content ​

  1. Interface and Function Optimization

    • Default enabled newly designed Liii Theme, providing more modern visual experience
    • "Tools->Keyboard->Show Keyboard Keys" supports Windows system
    • LaTeX Export Improvement: Default disabled macro customizations with tm prefix, output more standard LaTeX code

  2. Large Language Model Function Enhancement

    • Added support for SiliconFlow Kimi K2 Model, expanding dialogue capabilities
    • Improved mathematical symbol support: Optimized parsing and rendering of \oiint (loop double integral), \iiint (triple integral), \oint (loop integral)
    • macOS Convenience: No need to manually install Pandoc to use large language model dialogue
    • Windows Experience Fix: Resolved issue of black terminal window popping up when calling large language models

Link: Key Points Overview

Liii STEM v2025.0.11 (20250629) ​

Update Content ​

  1. Comprehensively improved user interface theme visual effects, added Liii user interface theme (users can switch themes in Edit -> Preferences -> User Interface Theme -> Liii)
  2. Fixed defect where VLM could not ask follow-up questions
  3. Fixed defect where switching tabs would return to document beginning instead of last mouse hover position
  4. Now supports Ctrl/Cmd + number (1-6) to switch tabs
  5. Now supports Ctrl/Cmd + T to add new tabs
  6. LLM plugin added support for models:
  • Qwen2.5-VL-7B-Instruct (supports VLM, small and fast, can be used for formula recognition, recommended prompt "recognize but don't reason")
  • Qwen2.5-VL-32B (supports VLM)
  • Qwen2.5-VL-72B-Instruct (supports VLM)
  • Qwen3-235B-A22B
  • MiniMax-M1-80k

Liii STEM v2025.0.10 (20250607) ​

Update Content ​

  1. Fixed defect where VLM couldn't load large images, can display as shown in the image effect
  2. Mathematical environment * now prioritizes displaying ×\times×
  3. Improved plugin loading method, significantly optimized performance

Attachment: VLM Function Example






Liii STEM v2025.0.8 (20250425) ​

Update Content ​

  1. Ubuntu 22.04 installation package (seed user zhengyq, Zeyu Dong requirement)
  2. Improved error handling to enhance LLM plugin user experience
  3. Supports VLM, enabling image AI Q&A, handwritten formula recognition, etc. (almost all users require)
  4. Fixed Definition/定义 environment to upright font (seed user iamspuerboy, Zeyu Dong, etc. requirement)
  5. Added Assumption/假设 environment (seed user HaggittLi, Cauchy, etc. requirement)
  6. Fixed defect where images couldn't be imported (discovered by sprouting user Ziegward)






Liii STEM v2025.0.7 (20250416) ​

Update Content ​

Function Fixes ​

  1. Fixed format exception issues when importing or pasting HTML content
  2. Fixed issue where file names containing asterisk(*) characters caused software unresponsiveness
    • Software will now prohibit creating or opening file names containing asterisks

LLM Function Enhancement ​

  1. Added proxy configuration support
    • Added proxy parameter for configuring network proxy
  2. Added model parameter adjustment
    • Supports adjusting temperature parameter to control output randomness
  3. Extended model support
    • Added support for OpenAI series large language models

  2. Now when modifying interface language, software automatically restarts to avoid software crash
  3. Supports exporting formatted Mogan Scheme
  4. Upgraded to Goldfish Scheme v17.11.9
