Slides ​

Mogan's slides are WYSIWYG versions of LaTeX Beamer slides. The main difference from Microsoft PowerPoint (PPT) is the target user group.

This part is exactly the same as document editing, please refer to previous help documentation

1.4 Adding Borders to Theorem and Other Enunciation Environments ​

First enter the enunciation that needs background rendering, then click Focus -> Preferences -> Theorem Border in the toolbar. This allows you to add borders to enunciations. This operation makes theorem and other enunciation environments visually independent from other text content. (Operation method and effect before and after operation, as shown below)

1.5 Distinguishing Between Collapsing Environments and Layers ​

Collapsing is essentially a special layer combination with interactive logic. Layers as basic carriers can support infinite hierarchical stacking, achieving element hierarchical management; while collapsing environments only consist of "expanded layer" and "collapsed layer" two layers, which are lightweight forms of layers in specific scenarios.

  • Inserting Collapsing Environments:

    • Inserting Already Collapsed

      • Function of already collapsed: Directly generates a component that can switch between collapsed or expanded states by clicking.

      • In the toolbar, click Toggle and Collapse -> Already Collapsed -> Default to insert a collapsing environment, click the left dot to switch state.

    • Inserting Switch

      • Function of switch: Without increasing the number of slides, enables local content to change effects, better assisting our presentation. (Can be understood as a large slide containing some small slides).

      • Switch insertion method: In the toolbar, click Insert -> Fold -> Switch. (At this time, you can choose "Standard" or "Small", where "Standard" is similar to inserting a "displayed formula", "Small" is similar to inserting an "inline formula".) (Insertion path and effect as shown below)

    • Inserting Unroll

      • Function of unroll: Without increasing the number of slides, enables related content to be displayed sequentially in a list-like form, thus better assisting our presentation.

      • Unroll insertion method: In the toolbar, click Insert -> Fold -> unroll. (At this time, you can choose "Compress", "Phantoms" or "Greyed", where "Compress" will hide subsequent text without leaving space, "Phantoms" will hide subsequent text but retain space, "Greyed" will gray out subsequent text and retain space) (Insertion path and effect as shown below)

  • Inserting Layers: In the menu bar, click Insert -> Fold -> Layer -> Standard, you can insert multiple layers through structured insertion. When clicking the layer position, the current layer display will appear in the environment toolbar and you can directly switch layers.

    After creating layers, all content will be displayed by default. Therefore, to achieve the function of each layer displaying its own content separately, you need to select the content in the layer to be changed and then click Insert -> Fold -> Layer -> Visible only here in the menu bar to achieve the effect of only visible in the current layer. You can also achieve other functions according to this step, such as clicking Insert -> Fold -> Layer -> Visible from here on to achieve the function of displaying from current layer and subsequent layers; clicking Insert -> Fold->Layer->Visible except here to achieve the function of not displaying except current layer.

3 How to Present Slides ​

3.1 Page Turning ​

Next (PageDown) means the next object rather than by page unit. Clicking its function is similar to a page turner that can select consecutive next objects.

3.2 Viewing Modes ​

There are multiple modes to choose from in the menu bar's View. When Focus Mode and Presentation Mode are enabled, the top menu bar and toolbar will disappear. Right-click to uncheck the mode to restore default state.
