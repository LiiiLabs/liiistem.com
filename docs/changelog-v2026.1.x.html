Liii STEM v2026.1 Version Change Log ​

⭐️ New Features and Highlights ​

  1. Math OCR + Magic Paste: quickly recognize formulas from images/screenshots/handwriting and even PDF papers, then paste directly as Liii STEM format (with dedicated shortcuts for Text/Math modes).

  1. Selective Paste: automatically detects clipboard formats and lets you paste as Markdown / LaTeX / HTML, etc. (Ctrl+Alt+V / Command+Option+V).

  1. More AI models: added GLM-4.6V and Kimi-K2.5, improving AI-assisted writing and chat.

  1. Better tables: drag internal grid lines with the mouse; added three-line tables for more efficient academic table layout.

  1. Structured search & replace: search/replace structured content including text, formulas, symbols, and matrices.

  1. New comments/annotations: add foldable comments from the menu bar; comments can contain formulas and other structured content (not just plain text).

  1. More syntax highlighting: added R / SQL / Bash syntax highlighting.
  2. Academic writing support: references now support GBT7714-2015.

🌟 Summary ​

Liii STEM v2026.1 continues to refine Magic Paste: faster Math OCR and richer paste options (including Markdown/LaTeX/HTML), plus stronger tables (gridline dragging and three-line tables), expanded AI model support, and a broad set of stability and usability fixes.

🔧 Changelog ​

Liii STEM v2026.1.8 (2026.03.29) ​

  1. Further improved OCR: supports more titles, lists, HTML, LaTeX, and figure-caption content, with more accurate structure recovery and more stable image insertion.
  2. Fixed formula numbering and several layout issues, including aligned, section, and appendix-related cases.

Liii STEM v2026.1.7 (2026.03.18) ​

  1. Added JSON syntax highlighting.
  2. Improved Python syntax highlight colors in dark mode.
  3. Added support for editing the author name for the annotation focus toolbar.
  4. Fixed a crash issue in the macro editor.
  5. Fixed several Bugs.

Liii STEM v2026.1.6 (2026.03.03) ​

  1. LaTeX import now supports \bold and \italic.
  2. Improved table border colors in dark mode.
  3. Fixed several Bugs.

Liii STEM v2026.1.5 (2026.02.09) ​

  1. Replaced some Math Mode toolbar icons.
  2. Added support for Intel-based macOS systems.

Liii STEM v2026.1.4 (2026.02.06) ​

  1. AI: added new models GLM-4.6V and Kimi-K2.5.
  2. Added three-line table type: Insert → Table → Three-line table.
  3. Fixed a reference issue that caused Math completion Tab Cycling pop-ups to appear in the wrong position.
  4. Fixed misaligned layout when importing LaTeX piecewise functions.
  5. Adjusted the UI of the “Open PDF” dialog.
  6. On Windows, prefer the app’s built-in Pandoc.

Liii STEM v2026.1.3 (2026.01.30) ​

  1. Fixed a Python plugin error caused by sympy.symbol.
  2. Removed an incomplete Octave plugin that could freeze the app.
  3. When opening existing files, removed the extra “Untitled” label that could appear in the title bar.

Liii STEM v2026.1.2 (2026.01.23) ​

  1. Added SQL syntax highlighting.
  2. Added Bash syntax highlighting.
  3. Fixed Tab Cycling position issues when zooming.
  4. Fixed tearing artifacts when rendering large radical signs.

Liii STEM v2026.1.1 (2026.01.22) ​

  1. Magic Paste enhancements: supports lossless paste from web pages/chat content; Text Mode supports image OCR; Math Mode supports LaTeX → Liii STEM conversion.
  2. Selective Paste: choose to paste as Markdown / LaTeX / HTML, etc., including image OCR to formulas.
  3. Structured search & replace: supports structured content such as text, formulas, symbols, and matrices.
  4. New comment/annotation feature: fold/unfold comments; comment content supports text, formulas, and more.
  5. Dark mode: both UI and documents support dark mode.
  6. Typesetting engine optimization: Chinese punctuation compression.
  7. Table enhancements: drag internal grid lines with the mouse.
  8. Tab Cycling enhancements: added mathcal / mathbf / mathfrak / mathscr / mathbbm variants; added a \nabla variant for n.
  9. Image hover menu: quick layout adjustment and OCR recognition.
  10. nbsp display optimization: non-breaking spaces use an invisible color by default (adjustable).
  11. Added R syntax highlighting.
  12. References now support GBT7714-2015.
